# Changelog

## 0.0.1

- Initial release.
- Add Activity Bar launcher view for Codex, Claude Code, OpenCode, and Pi.
- Launch CLI TUI tools in VS Code integrated terminals.
- Open launched terminals in the editor area by default.
- Add configurable custom launchers.
- Add editor and Explorer commands for copying file and selection references.
- Support remote workspace execution for WSL, SSH, and Dev Containers.
